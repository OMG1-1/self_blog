# 周总结

## 记录规则

### 工作记录

- 记录周度重点完成的工作
- 记录周度工作的难点和解决方案
- 记录周度工作的收获和成长

### 学习记录

- 记录本周学习的重点和难点
- 记录本周学习的收获和成长

### 生活记录

- 记录本周生活的重点和难点
- 记录本周生活的收获和成长

## 记录形式

### 工作记录

- 工作内容
- 工作难点
- 解决方案
- 收获
- 成长

### 学习记录

- 学习内容
- 学习难点
- 收获
- 成长

### 生活记录

- 生活内容
- 生活难点
- 收获

## 记录时间

- 工作记录：周五晚 18:20
- 学习记录：周天晚 18:20 - 周天晚 22:00
- 生活记录：周天晚 19:00
