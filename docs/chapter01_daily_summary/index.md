# 日总结

## 记录时间

- 工作记录时间：每个工作日下午 18:00
- 学习记录时间：晚上 20:00 - 晚 22:00

## 记录内容

### 工作记录

记录每天的工作内容, 工作收获, 工作问题, 工作思考, 工作计划

### 学习记录

记录每天的学习内容, 学习收获, 学习问题, 学习思考, 学习计划
